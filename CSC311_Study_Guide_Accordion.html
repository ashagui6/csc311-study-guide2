Format: MCQs, T/F, Fill-in-the-blank. Focused on problem solving using lab machines and lockdown browser.

Topic 1: Algorithms

- Understand time complexity (O(n), O(log n), etc.)
- Be able to analyze pseudocode
- Smaller complexities are included in larger ones

Topic 2: Arrays / ArrayList

- ArrayList grows dynamically
- Insertions/deletions involve shifting
- Implementation knowledge

Topic 3: Stack

- Operations: push, pop, peek, isEmpty
- Applications: reverse string, balanced parentheses, postfix evaluation

Topic 4: Linked List

- Single, double, circular, corrupt lists
- Time complexity of operations

Topic 5: Queue

- Basic operations and complexities
- Circular queue advantage

Topic 6: Recursion

- Base case
- Examples: Factorial, Fibonacci, Towers of Hanoi

Topic 7: Sorting

- Understand sorting steps for one algorithm

Topic 8: Hash Tables

- Insert/delete operations
- Collision handling: linear probing & chaining
- Be able to show final hash table state

Topic 9: Trees

- BST operations and traversals
- AVL tree rotations
- 2-3 trees and skip lists (basic understanding)
- Min/Max Heap insert/delete

Topic 10: Graphs

- Directed vs. undirected
- Representations: Adjacency list/matrix
- Traversals: BFS, DFS, BFS Tree
